### Tecnologias utilizadas

* Maven
* Spring Boot
* H2

### Para rodar este projeto localmente
```
$ git clone https://github.com/jaspm2004/ProductAPI
$ mvn clean install
$ cd target
$ java -jar .\ProductAPI-1.0.0.jar
```
API REST fica em http://localhost:8080/productapi/products
